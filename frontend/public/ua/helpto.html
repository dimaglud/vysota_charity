Кому ми допомагаємо / види допомоги
1. Спортсменам // Альпіністам, бігунам, велосипедистам, скелелазам, трейлранерам.
* Під спортсменом розуміємо особу, що має ліцензію від відповідної федерації.
  • Спорядження для участі у змаганнях та тренувань
  • Фінансування участі в змаганнях
  • Фінансування тренувань
  • Безповоротна фінансова допомога на інші потреби

2. Клубам // Альпіністським, біговим, велосипедним, скелелазним, спелеологічним, триатлоністи.
* Клубом називаємо організації, що налічує хоча б 4 спортсмени, та офіційно зареєстрована, або має сторінки в соціальних мережах.
  • Спорядження учасникам для тренувань
  • Обладнання для облаштування тренувальних зон
  • Обладнання для брендування
  • Фінансова допомога для закупівлі спорядження
  • Фінансова допомога для закупівлі обладнання
  • Фінансування закордонних поїздок та зборів
3. Окремим залам/тренувальним майданчикам, в тому числі їх працівникам // Скелелазним залам, станціям прокату водного спорядження, легкоатлетичним залам, велосипедним трекам та залам, байкпаркам,  школам з бігових лиж тощо, приміщення чи матеріальний фонд яких постраждав в результаті бойових дій, або був повністю втрачений, через окупацію.
  • Обладнання для облаштування тренувальних зон
  • Фінансування для ремонту/модернізації залів
  • Фінансування заходів на території залів
  • Фінансова допомога працівникам/тренерам

4. Збірним командам // Скелелазам, біговим командам, збірним з дуатлону чи бігових лиж, збірним з гірськолижного спорту та сноуборду, велосипедним командам та атлетам тощо.
* Команда повинна мати офіційний статус та відповідно бути зареєстрована у своїй федерації
  • Спорядження для участі у змаганнях та тренувань 
  • Фінансування закордонних поїздок та зборів
  • Фінансування участі в змаганнях

5. Родичам спортсменів // Допомога родичам спортсменів (п.1), що служать в армії України, отримали поранення на фронті чи постраждали в результаті бойових дій або загинули
  • Спорядження для повсякденного використання
  • Оплата відпочинку чи реабілітації дітей за кордоном 
  • Безповоротна фінансова допомога на інші потреби
6. Магазинам // Дилерам туристичного та спортивного спорядження, що постраждали від бойових дій (втратили частину товару, отримали пошкодження приміщення чи обладнання або взагалі були змушені покинути свій магазин в окупації).
  • Обладнання для брендування
  • Фінансування для ремонту/модернізації
  • Спорядження для повсякденного використання
7. Змагання // Скелелазні, бігові, змагання з трейлрану, скандинавської ходи, бігових лиж, гірськолижного спорту чи сноуборду, велосипедні змагання та веломарафони.
*  Під змаганнями розуміємо аматорські змагання для всіх бажаючих або професійні змагання, що містяться в календарному плані відповідної федерації
  • Товари відповідного бренду для призового фонду
  • Фінансова допомога на проведення змагань
  • Фінансування призового фонду
  • Благодійний внесок на збір, який проводять організатори подій
